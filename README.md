## 課題2概要
<br>本プログラムはROSを用いてLEDを光らすものである.
<br>Publisher側では入力を受け付け, Subscriber側ではLEDの状態を表示することができる.
## 操作方法
  1. 起動<br>
  ```
    $ roslaunch mymkg flash.launch
    $ [0, 1]
    $ [LED OFF, LED OFF]
  ```
